# Charlie Blackjack Plugins

## Overview

This repository contains plugins developed for the **Charlie Blackjack framework**. The goal of this project is to extend the base Charlie software by adding new gameplay functionality through a **plugin-based architecture**.

The project is currently a **work in progress**. The long-term objective is to implement several plugins that enhance gameplay and analysis of blackjack hands.

One component currently under development is a **Basic Strategy advisor**, which will eventually recommend optimal plays based on the player's hand and the dealer’s upcard.
